### ✅ `files: List[UploadFile] = File(...)`

这行代码的含义是：

#### 1. **`files: List[UploadFile]`**
- 声明参数 `files` 是一个 **列表（List）**，其中每个元素都是 **`UploadFile` 类型的对象**。
- `UploadFile` 是 FastAPI/Starlette 提供的一个类，用于表示上传的文件，支持：
  - 异步读取（`.read()`）
  - 获取文件名（`.filename`）
  - 获取 MIME 类型（`.content_type`）
  - 文件内容可能存储在内存或临时文件中（对大文件友好）

#### 2. **`= File(...)`**
- 这是 FastAPI 的**依赖注入语法**，用来告诉框架：
  > “这个参数不是普通的 JSON body 或 query 参数，而是来自 HTTP 请求中的 **表单文件字段（multipart/form-data）**。”
- `File(...)` 中的 `...`（Python 的 `Ellipsis`）表示**该参数是必需的（required）**。
  - 如果写成 `File()`（无参数），则表示**可选**（默认值为 `None` 或空列表）。
  - 如果写成 `File(default=...)`，其实和 `File(...)` 等价。

> 🔸 所以：`File(...)` 的作用有两个：
> 1. **指定参数来源**：来自上传的文件（不是 JSON、不是 query）。
> 2. **标记为必填**：客户端必须上传至少一个文件，否则会返回 422 Validation Error。

---

### 🧪 举个例子

```python
@app.post("/upload")
async def upload(files: List[UploadFile] = File(...)):
    ...
```

- ✅ 合法请求：`POST /upload`，body 为 `multipart/form-data`，包含一个或多个 `<input type="file" name="files">`。
- ❌ 非法请求：
  - 不传任何文件 → FastAPI 返回 422 错误：“field required”
  - 传 JSON 而不是 form-data → 422 错误

---

### 🔄 对比其他写法

| 写法 | 含义 |
|------|------|
| `files: List[UploadFile] = File(...)` | 必须上传 ≥1 个文件，每个是 `UploadFile` |
| `files: List[UploadFile] = File()` | 可选上传，不传时 `files = []`（空列表） |
| `files: List[bytes] = File(...)` | 必须上传文件，但内容直接作为字节读入内存（适合小文件） |

> ⚠️ 注意：`List[UploadFile]` 和 `List[bytes]` 是两种不同的处理方式，不能混用 `.read()`。
